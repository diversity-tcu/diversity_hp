# イベント情報を追加する

---

## 手順

### 1. ファイルを作成する

`src/content/events/` フォルダに新しい `.md` ファイルを作成します。

- **ファイル名**: `YYYY-MM-DD.md`（イベントの日付）
- 同じ日に複数のイベントがある場合: `YYYY-MM-DD-2.md`, `YYYY-MM-DD-3.md`

### 2. テンプレートをコピーして編集する

[テンプレート](templates/event.md) の内容をコピーし、以下を編集してください:

| 項目 | 説明 | 例 |
|------|------|----|
| `title` | イベントタイトル | `"第10回ダイバーシティフォーラム"` |
| `date` | 日付（YYYY-MM-DD） | `"2026-06-15"` |
| `topic` | カテゴリ（任意） | `"environment"`, `"reform"`, `"nextgen"` |
| 本文 | イベントの詳細（Markdown記法） | — |

### 3. 画像がある場合

1. 画像を `public/uploads/YYYY/MM/` に保存
2. 本文中で参照: `![説明](/uploads/YYYY/MM/filename.jpg)`

### 4. 公開する

→ [公開手順（commit & push）](publish.md)

## topic の選択肢

| 値 | 意味 |
|----|------|
| `environment` | 環境整備 |
| `reform` | 意識改革 |
| `nextgen` | 次世代育成 |

省略可能です。該当するカテゴリがあれば指定してください。

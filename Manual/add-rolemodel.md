# ロールモデル集の新号を追加する

卒業生ロールモデル集・教員ロールモデル集の追加手順です。

---

## 手順

### 1. 画像を配置する

画像ファイルを `public/uploads/YYYY/MM/` に保存します。

| 画像 | ファイル名の例 | 説明 |
|------|---------------|------|
| メイン写真 | `role19_01.jpg` | 一覧ページの書影にも使われる |
| スタイル画像 | `role19_style.png` | 1日の過ごし方の画像（任意） |
| 本文用写真 | `role19_02.jpg`, `role19_03.jpg` | 趣味・エピソード等の写真（任意） |

教員の場合は `t_role09_01.jpg` のように `t_` を付けて区別します。

### 2. ファイルを作成する

| 種別 | 保存先 | ファイル名 |
|------|--------|-----------|
| 卒業生 | `src/content/pages/publication/rolemodel/` | `vol-XX.md` |
| 教員 | `src/content/pages/publication/t_rolemodel/` | `t_volXX.md` |

### 3. テンプレートをコピーして編集する

- 卒業生: [テンプレート](templates/rolemodel.md)
- 教員: [テンプレート](templates/t_rolemodel.md)

HTMLの知識は不要です。frontmatter にプロフィール情報を記入し、本文は普通のMarkdownで書きます。

#### 卒業生の frontmatter

| 項目 | 必須 | 説明 | 例 |
|------|:----:|------|----|
| `title` | ○ | **接頭辞ルール必須**（下記参照） | `"社会で輝く卒業生たち Vol.19"` |
| `date` | ○ | 公開日 | `"2026-06-01"` |
| `name` | ○ | 氏名 | `"山田 花子"` |
| `affiliation` | | 所属（学部・学科） | `"理工学部 機械工学科"` |
| `role` | | 敬称 | `"さん"` |
| `photo_main` | | メイン写真 | `"/uploads/2026/06/role19_01.jpg"` |
| `workplace` | | 勤務先（改行は `\n`） | `"○○株式会社\n○○部"` |
| `education` | | 出身学部（改行は `\n`） | `"20XX年 東京都市大学 … 卒業"` |
| `high_school` | | 出身高校 | `"○○高等学校 卒業"` |
| `photo_style` | | 1日のスタイル画像 | `"/uploads/2026/06/role19_style.png"` |

#### 教員の frontmatter

| 項目 | 必須 | 説明 | 例 |
|------|:----:|------|----|
| `title` | ○ | **接頭辞ルール必須**（下記参照） | `"東京都市大学の多様な教員たち Vol.09"` |
| `date` | ○ | 公開日 | `"2026-06-01"` |
| `name` | ○ | 氏名 | `"田中 太郎"` |
| `affiliation` | | 所属（学部・学科） | `"理工学部 機械工学科"` |
| `role` | | 役職 | `"教授"` |
| `photo_main` | | メイン写真 | `"/uploads/2026/06/t_role09_01.jpg"` |
| `theme` | | テーマの一言 | `"研究テーマを象徴する一言"` |
| `career` | | 経歴（改行は `\n`） | `"○○大学卒業。\n○○大学大学院修了。"` |
| `photo_style` | | 1日のスタイル画像 | `"/uploads/2026/06/t_role09_style.png"` |
| `published_date` | | 掲載日 | `"2026年6月1日"` |

### タイトルのルール（必須）

一覧ページのフィルタ条件に使われるため、接頭辞を正確に記載してください:

- **卒業生**: `社会で輝く卒業生たち Vol.XX`
- **教員**: `東京都市大学の多様な教員たち Vol.XX`

### 4. 公開する

→ [公開手順（commit & push）](publish.md)

## 書影（サムネイル）について

一覧ページに表示される書影は `photo_main` で指定した画像が使われます。
`photo_main` を省略した場合は本文中の最初の画像が自動的に使われます。どちらもなければデフォルト画像が表示されます。

## 注意事項

- 一覧は `date` の降順で自動表示されます
- 本文中に画像を含める場合は `![説明](/uploads/YYYY/MM/filename.jpg)` と書きます
- 既存号を参考にすると統一感が出ます
